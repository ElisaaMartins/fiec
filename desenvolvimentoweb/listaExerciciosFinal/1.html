Responda:

a) Qual a diferença entre entidade, atributo e tag?

R: Entidade = é o conjunto de caracteres usado para representar outros caracteres que são invisiveis ou dificeis de digitar;
Atributos = são palavras especiais usadas dentro das tags de abertura, que controlam o comportamento da tag;
Tag = são elementos do HTML, que informam ao navegador que tipo de estrutura está sendo criada.

b) No css, qual a diferença entre classe, tag e id?

R: Entidade = começa com & e termoina com ";" são usadas para criar simbolos como "& copy ; (©)";
Classes = é a forma de identificar um grupo de elementos, podendo atribuir estilos a vários elementos de uma só vez;
ID = é a forma de de identificar um elemento e é única para cada um;
Tag = são delimitadores de estilo ou de conteudo.
